# Agents

All project instructions live in `.github/`. Read them before editing code.

## Instruction files

| File | Applies to | Covers |
|------|-----------|--------|
| `.github/copilot-instructions.md` | Everything | Branch workflow, project shape, conventions, anti-patterns, CI |
| `.github/instructions/rails.instructions.md` | `**/*.rb`, `**/*.rake`, `Rakefile`, `Gemfile` | Rails and Ruby conventions |
| `.github/instructions/rspec.instructions.md` | `spec/**/*_spec.rb` | RSpec test conventions |
| `.github/instructions/frontend.instructions.md` | `frontend/**/*.ts`, `frontend/**/*.tsx` | TypeScript and React conventions |
| `.github/instructions/date-time.instructions.md` | `frontend/**/*.ts`, `frontend/**/*.tsx`, `**/*.rb` | Date/time handling rules |
